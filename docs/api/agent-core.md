# `@conscius/agent-core` (removed)

The former core package (context builder, plugin loader, hook runner) and CLI entry are split as of **v3**:

- **`@conscius/runtime`** — engine, types, hooks, memory pipeline
- **`@conscius/cli`** — `conscius` binary

→ See **[`@conscius/runtime`](./runtime.md)** and **[`@conscius/cli`](./cli.md)**.
